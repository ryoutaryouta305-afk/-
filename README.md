# 大富豪オンライン（地元ルール完全対応版）

2〜6人でブラウザだけで遊べるオンライン大富豪。革命・8切り・階段に加えて、地元ルール11種を実装。
見た目はシックなカジノの円卓風、必殺技発動時は専用の演出ムービーが流れる。

## 実装ルール
- 12ボンバー（Qをn枚→指定ランクを全員が捨てる／爆発演出）
- AK（A+K同時出し・場が空のとき→2枚渡して手番続行／銃撃演出）
- 7渡し（7をn枚→n枚を次の人へ）
- 10捨て（10をn枚→n枚を除外）
- Jバック（Jを出す→場が流れるまで革命反転）
- 救急車（9を2枚→墓地から1枚回収／救急車が画面を横切る演出）
- 報恩講（6+A+A）→相手の手札から1枚もらう・次はその人の番／演出あり
- 降誕会（5+A+2）→相手の手札から1枚もらう・次はその人の番／演出あり（報恩講と対になる釈迦モチーフ）
- ナイチンゲール（A+Q→墓地から1枚回収・発動後は自分の手番に強制）
- 都落ち（大富豪以外の人が新たに大富豪になった瞬間、前の大富豪は強制的に大貧民になる）
- カード交換（大富豪↔大貧民は2枚、富豪↔貧民は1枚／受け渡しモーションあり）
- ジョーカー上がり反則（ジョーカーを使い切って上がると強制的に最下位グループへ）
- 革命（4枚出し）・8切り・階段・ジョーカー

渡す枚数が足りないときは出せる分だけ渡し、手札が0枚になったらそのまま上がり。
ルールは部屋作成後にON/OFF切り替え可能。

## 演出について
AK・救急車・12ボンバー・報恩講・降誕会・ナイチンゲール・都落ち・ジョーカー上がり反則・カード交換のそれぞれで
専用の画面演出（技名の大きなテロップ＋固有アニメーション）が流れる。演出はCSSアニメーションで実装しており、
動画ファイルは使用していない。

## 遊び方（同じ端末で開発確認）
```
npm install
npm start
```
ブラウザで http://localhost:3000 を開く。

## みんなで遊ぶ手順
下の3つのどれか。

### A. 同じWi-Fi（家・同じ部屋にいる場合）
1. ホストのPCで `npm start`
2. ホストのローカルIPを調べる（Windows: `ipconfig` の IPv4 / Mac: `ifconfig | grep inet`）
3. 友達は同じWi-Fiで `http://（そのIP）:3000` を開く

### B. インターネット経由（離れた友達と）
無料のトンネルサービスで一時的に公開できる。
```
npm start            # 別のターミナルで起動しておく
npx localtunnel --port 3000
```
表示された `https://xxxx.loca.lt` を友達に共有。
（cloudflaredやngrokでも可）

### C. ちゃんと常設で公開（おすすめ）
Render / Railway などにデプロイ。GitHubに上げて連携すれば無料枠で動く。
Renderの場合: New → Web Service → リポジトリ選択 → Build `npm install` / Start `npm start`。

## 対戦の流れ
1. 名前を入れて「部屋を作る」→ 部屋コード（4文字）が出る
2. 友達はそのコードで「参加する」
3. 部屋主がルールを選んで「ゲーム開始」
4. カードをタップして選び「出す」/「パス」
5. 全員上がったら結果表示 → 部屋主が「次のラウンドへ」（カード交換フェーズを経て次ラウンド開始）
