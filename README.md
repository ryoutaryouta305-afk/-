# 大富豪オンライン（地元ルール完全対応版）

2〜5人でブラウザだけで遊べるオンライン大富豪。革命・8切り・階段に加えて、地元ルール10種を実装。

## 実装ルール
- 12ボンバー（Qをn枚→指定ランクを全員が捨てる）
- AK（A+K同時出し・場が空のとき→2枚渡して手番続行）
- 7渡し（7をn枚→n枚を次の人へ）
- 10捨て（10をn枚→n枚を除外）
- Jバック（Jを出す→場が流れるまで革命反転）
- 救急車（9を2枚→墓地から1枚回収）
- 釈迦（5+A+2）／報恩講（A2枚+6）→相手の手札から1枚もらう・次はその人の番
- ナイチンゲール（9+A→墓地から1枚回収）
- 都落ち（前回大貧民が1位→1位と最下位を交代）
- カード交換（大富豪↔大貧民は2枚、富豪↔貧民は1枚）
- 革命（4枚出し）・8切り・階段・ジョーカー

渡す枚数が足りないときは出せる分だけ渡し、手札が0枚になったらそのまま上がり。
ルールは部屋作成後にON/OFF切り替え可能。

## 遊び方（同じ端末で開発確認）
```
npm install
npm start
```
ブラウザで http://localhost:3000 を開く。

## みんなで遊ぶ手順
下の3つのどれか。

### A. 同じWi-Fi（家・同じ部屋にいる場合）
1. ホストのPCで `npm start`
2. ホストのローカルIPを調べる（Windows: `ipconfig` の IPv4 / Mac: `ifconfig | grep inet`）
3. 友達は同じWi-Fiで `http://（そのIP）:3000` を開く

### B. インターネット経由（離れた友達と）
無料のトンネルサービスで一時的に公開できる。
```
npm start            # 別のターミナルで起動しておく
npx localtunnel --port 3000
```
表示された `https://xxxx.loca.lt` を友達に共有。
（cloudflaredやngrokでも可）

### C. ちゃんと常設で公開（おすすめ）
Render / Railway などにデプロイ。GitHubに上げて連携すれば無料枠で動く。
Renderの場合: New → Web Service → リポジトリ選択 → Build `npm install` / Start `npm start`。

## 対戦の流れ
1. 名前を入れて「部屋を作る」→ 部屋コード（4文字）が出る
2. 友達はそのコードで「参加する」
3. 部屋主がルールを選んで「ゲーム開始」
4. カードをタップして選び「出す」/「パス」
5. 全員上がったら結果表示 → 部屋主が「次のラウンドへ」
